Initialize Firestore

In order to use MkTurk, two collections in Firestore need to be initialized. Pressing the "Initialize Firestore" button below will setup two collections:

  • devices
  • mkturkdata
The document in mkturkdata is not needed to run MkTurk but will show you, as an example, the data output of MkTurk.
